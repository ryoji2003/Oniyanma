# 博物館検定フェスアプリ 

博物館の展示物を題材に、AIと協力してクイズを作成・投稿し、来場者同士で解き合うことができるWebアプリケーションです。
AIが生成したクイズを人間が修正・確認してから投稿する、という設計を取り入れています。

## 実行環境
* Java Development Kit (JDK) 11以上
* IntelliJ IDEA (推奨)
* インターネット接続 (OpenAI APIを利用するため必須)

## 起動手順

### 1. サーバーの起動
IntelliJ IDEA上で以下のファイルを実行（Run）してください。
* **実行ファイル:** `src/main/java/jp/ac/u_aizu/quizapp/ThemeController.java`

コンソールに `Server started: http://localhost:8080` と表示されれば準備完了です。
※デモ終了まで停止しないでください。

### 2. アプリへのアクセス
ブラウザ（Chrome等）を開き、以下のURLにアクセスしてください。
* **URL:** [http://localhost:8080/](http://localhost:8080/)

⚠️ **注意:** ファイルを直接開くのではなく、必ず上記URLからアクセスしてください（API通信のため）。

---

## 操作マニュアル

### 1. 管理者設定（最初にやること）
1. アプリ下部のフッターにある **「Staff Only」** をクリックします。
2. パスワード **`admin`** を入力します。
3. **「現在のフェステーマ」** を入力し（例：農業、祈りなど）、「テーマを保存する」を押します。
4. 設定が終わったら、左上の「博物館検定フェス」ロゴなどを押してトップに戻ります。

### 2. クイズを作る（一般参加者）
1. トップ画面の **「作問に挑戦する」** をクリックします。
2. 時代と展示品を選択し、「この資料で問題を作る」ボタンを押します。
3. AIがクイズを生成します。
4. **【重要】編集モード:** AIが作った問題に間違いがあれば、**タップして手動で修正**できます。
5. **「修正して投稿」** を押すと、クイズがリストに保存されます。
